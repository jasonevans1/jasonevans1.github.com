Active Projects

Sep 5th, 2011

Below are the github projects I have worked on in the past 3 months.

Projects

Aoe Scheduler

This is a fork of the Magento extension that adds a scheduler interface to Magento’s admin panel. More info on Aoe_Scheduler.

I have forked this extension to support adding and updating tasks. I have also added support for setting optional parameters with a task. I have submitted a pull request to Fabrizio Branca. He has asked from some changes that I will be implementing in the coming weeks.

I think this extension will provide a needed feature in Magento. If you have any suggestions or comments please send me a message in github. Here is my github profile @jasonevans1

Vagrant Magento

I created this project to learn Vagrant and Puppet. Since I do Magento development daily, I thought it would be useful to have a Vagrant box with Magento installed. This project provides a foundation for developing with Magento Commerce in a Vagrant box. The machine is based on CentOS 5.5. It contains puppet modules for provisioning a server with Magento installed.
List of installed software. * Magento Community Edition - 1.5.1.0 * Mysql * Zend Server Community Edition - PHP 5.3 * xdebug * phpunit Note: I intend to update this project to install the latest version of Magento with 1.6.

iodocs

This is a fork of the iodocs project from Mashery. I have added support for basic username and password auth.

I forked this project to learn more about node.js. I am very interested in learning more about node.js.
